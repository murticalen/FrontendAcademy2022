Different tags have different default browser styles

Such as:

  • Font size and weight (bold)
  • Margins
  • Paddings
  • Display property

Note that header and footer tags render only text with no styles (separators are new tag), while other have distinct styling . Styling of non styled elements is left to developer, so fair question would be: 

Why should we use those tags when their layout is the same as for the div tag?

It's because our HTML is in most cases viewed by users using ordinary browsers, but specific visitors could also visit our page. Those specific visitors include Crawlers (Such as Google crawler) or users using screen readers. Specific tags help those users to navigate pages.
